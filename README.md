# Flight Tracker

A real-time flight tracking application built with React, TypeScript, and Vite.

## Features
- Track live flights over major Indian cities
- Interactive map view with Leaflet
- Mobile-friendly responsive design
- Real-time aircraft positions and details

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Leaflet (for maps)
- OpenSky Network API

## Installation
```bash
npm install
npm run dev
```

## Supported Cities
Bangalore, Mumbai, Delhi, Chennai, Kolkata, Hyderabad, Pune, Ahmedabad, Kochi, Goa
